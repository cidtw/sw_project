## 🛠️ 단계별 상세 워크플로우 (Workflow Steps)

### 1️⃣ 1단계: 수집 경로 분류 및 데이터 수집 (Data Ingestion)
안티 크롤링(Anti-Crawling) 차단벽을 우회하고 시스템의 안정성을 확보하기 위해 수집 경로를 이원화하여 전처리되지 않은 **Raw 데이터**를 확보합니다.

* **공식 API 활용 (안정적 백본 데이터):**
    * **사람인 Open API** 및 **워크넷(Worknet) API**를 호출하여 대기업/중견기업의 공고 리스트를 안정적으로 수집합니다.
* **웹 API 역추적 및 크롤링 (유연한 확장):**
    * **원티드(Wanted), 리멤버(Remember)** 등 CSR 기반 플랫폼의 내부 API 엔드포인트를 역추적하여 JSON 데이터를 추출합니다.
    * 필요시 봇 차단 방지를 위해 `Playwright-Stealth` 엔진 및 주거용 프록시(Residential Proxy) IP 로테이션을 결합합니다.
* **수집 데이터 스키마 (Raw JSON):**
    ```json
    {
      "company": "㈜포스코",
      "title": "2026년 포스코 AI 전문인력 채용",
      "details": "데이터엔지니어 | 신입·경력 | 대졸↑ | 서울, 경북",
      "deadline": "~2026.07.05(일)",
      "detail_url": "https://..."
    }
    ```

### 2️⃣ 2단계: 자동화 파프라인 트리거 및 외부 데이터 결합 (Orchestration)
**Activepieces**를 오케스트레이터로 활용하여 인간의 개입(Human Intervention)을 최소화하는 완전 자동화 루프를 트리거하고 데이터를 심층 확장합니다.

* **Webhook 트리거:** 1단계에서 수집된 Raw JSON이 생성되는 즉시 Activepieces의 Webhook 인터페이스로 데이터가 전달됩니다.
* **외부 데이터 매핑 (Data Enrichment):**
    * 수집된 `company`명을 식별자로 사용하여 **금융감독원 DART API** 및 **국민연금 가입자 추이 공공 데이터**와 매핑합니다.
    * 기업 규모(대기업/중견/스타트업), 산업군 분류, 최근 고용 성장 추이(입·퇴사율) 데이터를 기존 공고 정보에 결합합니다.

### 3️⃣ 3단계: AI 데이터 정형화 및 맞춤형 스코어링 (LLM & Scoring Engine)
**Dify.ai** 플랫폼을 브레인으로 삼아 **ChatGPT(OpenAI)** 엔진의 고도화된 컨텍스트 분석 기능을 수행합니다.

* **고정 변인(유저 프로필) 대조:**
    * Dify Knowledge Base(지식 베이스)에 저장된 서비스 이용 유저의 배경 정보(개인 커리어, 기술 스택, 자소서 등)를 컨텍스트로 불러옵니다.
* **ChatGPT 연산 및 정형화:**
    * **JD 3줄 요약 및 비전 추출:** 공고의 상세 직무기술서(JD)와 기업 보고서를 기반으로 '기업의 중장기 경영계획 핵심 키워드'를 도출합니다.
    * **Fit-Score 산출:** 유저 프로필과 채용 조건 간의 정합성을 분석하여 **코사인 유사도(Cosine Similarity) 기반의 개인 맞춤 적합도 점수**를 계산합니다.
    * **Structured Outputs:** OpenAI의 JSON Mode를 강제하여 사전에 정의된 정형 데이터 포맷으로 오차 없이 출력합니다.

### 4️⃣ 4단계: 시각화 및 Slack 알림 송출 (Visualization & Delivery)
별도의 프론트엔드 및 이미지 생성 서버 구축 없이 **Slack 인프라**를 극대화하여 모바일 및 PC 환경에 최적화된 대시보드를 제공합니다.

* **Slack Block Kit 변환:**
    * Activepieces가 Dify로부터 전달받은 최적화 JSON 데이터를 Slack의 UI 프레임워크인 **Block Kit 포맷**으로 맵핑합니다.
    * 텍스트 가독성을 높이는 섹션, 적합도 점수를 나타내는 시각적 요소, `[지원하러 가기]` 등의 인터랙티브 버튼을 동적으로 조립합니다.
* **최종 발송:** Slack Webhook API를 통해 타겟 유저의 개인 슬랙 채널로 맞춤형 채용 스코어보드 메시지를 즉시 포워딩합니다.
